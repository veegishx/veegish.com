## The Details

This site is based on Zaggonaut, which is a retro-inspired theme for Astro, built using TypeScript, TailwindCSS, and Astro. I really love it so i kept this page as a testament to it.

## The Features

- Content Collections
- Dark & light mode
- Customizable colors
- 100 / 100 Lighthouse score
- Fully accessible
- Fully responsive
- Type-safe
